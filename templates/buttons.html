Default Buttons

Just add .btn-v class to the regular bootstrap button.

<button class="btn btn-v btn-primary">Primary</button>

Six color schemes are available - .btn-default, .btn-primary, .btn-success, .btn-info, .btn-danger and .btn-inverse.

Flat Buttons

Add .btn-flat class to make any button flat.

<button class="btn btn-v btn-flat btn-primary">Primary</button>

Wide Buttons

Add .btn-wide class to make any button wide.

<button class="btn btn-v btn-wide btn-primary">Primary</button>

Super Buttons

Add .btn-super class to make any button go super. These are bolder and wider than .btn-lg.

<button class="btn btn-v btn-super btn-primary">Primary</button>

Block Buttons

Add .btn-block class to make any button block sized.

<button class="btn btn-v btn-block btn-primary">Primary</button>

Disabled Buttons

Buttons can be disabled by adding disabled attribute.

<button class="btn btn-v btn-primary" disabled>Primary</button>

Active Buttons

Add .active class to change the state of the button to active.

<button class="btn btn-v btn-primary active">Primary</button>

Buttons With Icons

Icons (for e.g. Font Awesome) can be added to buttons.

<button class="btn btn-v btn-primary"><i class="fa fa-bug"></i> Primary</button>

Icon Sizes


.btn-xs, .btn-sm and .btn-lg can be used to make buttons extra small, small and large sized respectively.

<button class="btn btn-v btn-lg btn-default">Large</button>

Icon Buttons

Only icons can also be used in the buttons with .btn-icon class.

<button class="btn btn-v btn-icon btn-primary"><i class="fa fa-rocket"></i></button>

Social Buttons

.btn-facebook, .btn-twitter and .btn-google-plus classes can be used to for social buttons.

<button class="btn btn-v btn-facebook"><i class="fa fa-facebook"></i> Facebook</button>

<button class="btn btn-v btn-icon btn-twitter"><i class="fa fa-twitter"></i></button>

Button Toolbar

Enclose a group of buttons within .btn-group class to make a button group.

<div class="btn-group">
    <button class="btn btn-v btn-default">Button 1</button>
    ...more buttons here...
</div>

To make a different sized button block, add .btn-group-xs, .btn-group-sm or .btn-group-lg class to .btn-group to make a button group extra small, small or large respectively.

<div class="btn-group btn-group-lg">
    ...buttons here...
</div>

Sets of <div class="btn-group"> can be combined into a <div class="btn-toolbar"> for more complex components.

<div class="btn-toolbar" role="toolbar">
    <div class="btn-group">...</div>
    <div class="btn-group">...</div>
    ...more button groups here...
</div>

A .btn-group can be nested inside other as a dropdown menu.

<div class="btn-group">
        <button type="button" class="btn btn-v btn-default">1</button>
        <button type="button" class="btn btn-v btn-default">2</button>

        <div class="btn-group">
            <button type="button" class="btn btn-v btn-default dropdown-toggle" data-toggle="dropdown">
                Dropdown
                <span class="caret"></span>
            </button>
            <ul class="dropdown-menu">
                <li><a href="#">Dropdown link</a></li>
                <li><a href="#">Dropdown link</a></li>
            </ul>
        </div>
    </div>

For vertical variation of the same, use .btn-group-vertical instead of .btn-group in the parent.

Turn a button into a dropdown toggle with some basic markup changes.

<div class="btn-group">
        <button type="button" class="btn btn-v btn-default dropdown-toggle" data-toggle="dropdown">
            Default <span class="caret"></span>
        </button>
        <ul class="dropdown-menu" role="menu">
            <li><a href="#">Action</a></li>
            ...more dropdown links...
        </ul>
    </div>

Similarly, split button dropdowns can be created with the same markup changes, only with a separate button.

<div class="btn-group">
        <button type="button" class="btn btn-v btn-default">Default</button>
        <button type="button" class="btn btn-v btn-default dropdown-toggle" data-toggle="dropdown">
            <span class="caret"></span>
            <span class="sr-only">Toggle Dropdown</span>
        </button>
        <ul class="dropdown-menu" role="menu">
            <li><a href="#">Action</a></li>
            ...more dropdown links...
        </ul>
    </div>

Button dropdowns work with buttons of all sizes and can be set with .btn-xs, .btn-sm and .btn-lg.
