Default Buttons

Add .btn-v class to the regular bootstrap button.

<button class="btn btn-v btn-primary">Primary</button>

Six color schemes are available - .btn-default, .btn-primary, .btn-success, .btn-info, .btn-danger and .btn-inverse.

Flat Buttons

Add .btn-flat class to make any button flat.

<button class="btn btn-v btn-flat btn-primary">Primary</button>

Wide Buttons

Add .btn-wide class to make any button wide.

<button class="btn btn-v btn-wide btn-primary">Primary</button>

Super Buttons

Add .btn-super class to make any button go superman.

<button class="btn btn-v btn-super btn-primary">Primary</button>

Block Buttons

Add .btn-block class to make any button block sized.

<button class="btn btn-v btn-block btn-primary">Primary</button>

Disabled Buttons

Buttons can be disabled by adding disabled attribute.

<button class="btn btn-v btn-primary" disabled>Primary</button>

Buttons With Icons

Icons (for e.g. Font Awesome) can be added to buttons.

<button class="btn btn-v btn-primary"><i class="fa fa-bug"></i> Primary</button>

Social Buttons


.btn-facebook, .btn-twitter and .btn-google-plus can be used to for social buttons.

<button class="btn btn-v btn-facebook"><i class="fa fa-facebook"></i> Facebook</button>

<button class="btn btn-v btn-icon btn-twitter"><i class="fa fa-twitter"></i></button>

Icon Buttons

Only icons can also be used in the buttons with .btn-icon class.

<button class="btn btn-v btn-icon btn-primary"><i class="fa fa-rocket"></i></button>
