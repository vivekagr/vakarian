Buttons Groups

Large Sized

Button Toolbar

Enclose a group of buttons within .btn-group class to make a button group.

<div class="btn-group">
    <button class="btn btn-default">Button 1</button>
    ...more buttons here...
</div>

To make a different sized button block, add .btn-group-xs, .btn-group-sm or .btn-group-lg class to .btn-group to make a button group extra small, small or large respectively.

<div class="btn-group btn-group-lg">
    ...buttons here...
</div>

Sets of <div class="btn-group"> can be combined into a <div class="btn-toolbar"> for more complex components.

<div class="btn-toolbar" role="toolbar">
    <div class="btn-group">...</div>
    <div class="btn-group">...</div>
    ...more button groups here...
</div>

A .btn-group can be nested inside other as a dropdown menu.

<div class="btn-group">
        <button type="button" class="btn btn-default">1</button>
        <button type="button" class="btn btn-default">2</button>

        <div class="btn-group">
            <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
                Dropdown
                <span class="caret"></span>
            </button>
            <ul class="dropdown-menu">
                <li><a href="#">Dropdown link</a></li>
                <li><a href="#">Dropdown link</a></li>
            </ul>
        </div>
    </div>

For vertical variation of the same, use .btn-group-vertical instead of .btn-group in the parent.

Turn a button into a dropdown toggle with some basic markup changes.

<div class="btn-group">
<button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
    Default <span class="caret"></span>
</button>
<ul class="dropdown-menu" role="menu">
    <li><a href="#">Action</a></li>
    ...more dropdown links...
</ul>
</div>

Similarly, split button dropdowns can be created with the same markup changes, only with a separate button.

<div class="btn-group">
<button type="button" class="btn btn-default">Default</button>
<button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
    <span class="caret"></span>
    <span class="sr-only">Toggle Dropdown</span>
</button>
<ul class="dropdown-menu" role="menu">
    <li><a href="#">Action</a></li>
    ...more dropdown links...
</ul>
</div>

Button dropdowns work with buttons of all sizes and can be set with .btn-xs, .btn-sm and .btn-lg.
